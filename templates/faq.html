About

This is an open source project created to conduct sentiment analysis on the tweets of individuals as well as tweets regarding a specific subject. I created this project because I believe that Twitter is a valuable source of information regarding emotions, opinions and the current mental state of its users. Given that Twitter has millions of users of all backgrounds, it is the perfect resource for real-time global sentiment analysis. To contribute simply make a pull request on the github repository found here. Feel free to send me any suggestions.
